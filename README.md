# BNS Invariants Visualizations

This repository contains interactive visualizations related to the **Bieri-Neumann-Strebel (BNS)** invariants and related geometric group theory concepts.

## Projects

### [Geometric Lemma 1.1](geometric-lemma/index.html)

An interactive tool visualizing the condition for pulling points back into a ball using a finite collection of sets, based on:

> **Bieri, Robert; Strebel, Ralph.** Valuations and finitely presented metabelian groups.  
> *Proc. London Math. Soc. (3)* 41 (1980), no. 3, 439â€“464.  
> **MR0591649 (81j:20052)**

## Usage

1. Clone the repository.
2. Open `index.html` to see the list of available visualizations.
3. Navigate to specific tools (e.g., `geometric-lemma/index.html`) to use them.

## Deployment

To deploy this on GitHub Pages:

1. Push this repository to GitHub.
2. Go to **Settings > Pages**.
3. Select the `main` branch as source.
4. The site will be available at `https://YOUR_USERNAME.github.io/BNS/`.
